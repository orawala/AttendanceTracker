This is a simple web-based attendance tracker built using HTML, CSS, and JavaScript. The goal of this project was to create a lightweight and easy-to-use tool for students to keep track of their daily attendance without relying on any external apps or services.

Features

Add and manage subjects Mark attendance as Present, Absent, or Not Conducted Select number of classes per day Dashboard showing attendance percentage and required classes Calendar view to browse and edit past records Import and export data as JSON Works completely offline using local storage

How it Works

All the data is stored in the browser using localStorage, so nothing is sent to any server. You can open the file in any browser and start using it immediately. The calendar allows you to navigate between months and edit attendance for specific dates.

Usage

Open the HTML file in your browser Add your subjects in the Subjects tab Go to Mark and record your attendance Use Dashboard to monitor your percentage Use History to edit previous records if needed

Notes

Data will remain saved unless you clear your browser storage Export your data regularly if you don’t want to lose it This project is not intended for production use, just a personal tool
